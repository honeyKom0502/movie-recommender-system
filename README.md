# 🎬 Movie Recommender System

This is a content-based movie recommender system built using machine learning.

## 🚀 Features
- Data preprocessing
- Feature engineering
- Cosine similarity
- Movie recommendation

## 🛠 Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## 📂 Dataset
- TMDB 5000 Movies Dataset

## ▶️ How it works
- Converts movie data into vectors
- Calculates similarity between movies
- Recommends top 5 similar movies
